Javascript ES9 User Guide

Reinterpret and reorganize the MDN Javascript documentation for constructing a complete language usage model.

2019/05/01

0. Introduction

I have used Javascript for years now but realized I haven't been using it systematically as I did for C, C++, and Java. I'd like to know all possible language features, grammars, and idioms which can offer the same functionalities for a task to evaluate the advantages and disadvantages of each method. I truly believe this systematic approach makes me a better engineer.

To achieve that, I need a reference which contains all major and minor subjects of Javascript in a page like a cheatsheet. Each entry should be self-descriptive and have a link to its details for further research and reference.

The following is reinterpretation and reorganization of the famous Javascript reference made by MDN (Mozilla Developer Network). If you want to use it as a programming reference, I also recommend being familiarized with the ECMAScript Specification.

0. Conclusion

Javascript has been rapidly evolved to support the various purposes in the Web but seems like it is being stabilized recently as a fully-fledged programming language that has almost all features in the modern programming languages.

Personally, I think the future of Javascript is very bright since the Web is eating the world from traditional web pages to interfaces for IoT and AI applications. Long live Javascript!
