C - C++ - All in One Include File

A header file that contains all possible C/C++ standard libraries.

2018/08/01

0. Introduction

It is hard for a C/C++ programmer to manage a mental list of all possible libraries. Also, there are some situations that you simply don't care what you include, but you want to include all of them (prototyping or solving algorithm questions). There is a convenient header file called "<bits/stdc++.h>", which contains all include files, but sometimes it's not enough to include all of them blindly.

So I made this file. Thanks to the most famous C/C++ reference pages (cppreference.com and cplusplus.com) giving me the base information. I organized it and reformatted it so the user simply drag & copy & paste. Although this is a secondary document that could be protected by copyright laws, I don't claim any of rights. You can use it for any purposes within the range that doesn't violate the rights of the first writer (probably C/C++ standard committee). If you improve it, it would be very appreciated if you can share with me.

1. c_all_headers.hpp

/* c_all_headers.hpp
Title: C All Headers
Contents:
This collective header file is made to save time to include required header
files every time needed. The list is from the famous cppreference.com.
To use it from any cpp file, simply #include "c_all_headers.hpp".
*/
